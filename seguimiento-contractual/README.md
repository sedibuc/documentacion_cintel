# Micrositio de seguimiento contractual

Micrositio estatico para seguimiento del cumplimiento contractual de los proyectos RAG y Agente de Marketing / Agentes de IA, con foco en segunda fase.

## Paginas incluidas

- index.html
- alineacion-contractual.html
- segunda-fase.html
- matriz-entregables.html
- faltantes.html
- seguimiento-rag.html
- seguimiento-agent.html
- riesgos.html
- recomendaciones.html

## Objetivo del micrositio

Permitir lectura ejecutiva y tecnica de:

1. Exigencias contractuales.
2. Evidencia documental disponible.
3. Estado de cobertura por entregable.
4. Brechas y acciones de cierre.

## Ejecucion local

Abrir con servidor HTTP local desde la raiz del repositorio.

Ejemplo:

```bash
python -m http.server 8000
```

Luego abrir:

```text
http://localhost:8000/seguimiento-contractual/
```

## Nota de uso

Este micrositio prioriza trazabilidad contractual y no reemplaza la documentacion tecnica original de RAG ni agent. Funciona como capa de seguimiento de cumplimiento.
